Android device tree for the Lenovo Vibe K5/K5 Plus (A6020)
======================================================

The Lenovo Vibe K5/K5 Plus (codenamed _"A6020"_) is a mid-range smartphone from Lenovo mobility.
It was announced in February 2016.
